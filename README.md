## Description

Nest demo sucv.

## Project setup

```bash
$ pnpm install
```

## Compile and run the project

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## Run tests

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```

## docker
```bash
$ docker run --name sucursal -p 5432:5432 -e POSTGRES_PASSWORD=mysecretpassword -e POSTGRES_USER=antilef -v /data:/var/lib/postgresql/data -d postgres:17
```
